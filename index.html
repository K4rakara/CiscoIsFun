Quick Reference
Switch> enable Enters privileged EXEC mode.
Switch# configure terminal Enters terminal configuration mode.
Switch(config)# hostname foo Sets the hostname to "foo".
foo(config)# banner motd "Keep out" Sets the MOTD to "Keep out".
foo(config)# enable password baz Sets the unencrypted enable password to "baz".
foo(config)# enable secret baz Sets the encrypted enable password to "baz".
foo(config)# line con 0 Enters console line configuration mode.
foo(config-line)# password bar Sets the console line password to "bar".
foo(config-line)# login Enables a login screen.
foo(config-line)# exit Exits console line configuration mode.
foo(config)# line vty 0 15 Enters VTY line configuration mode.
foo(config-line)# password bar Sets the VTY line password to "bar".
foo(config-line)# login Enables a login screen.
foo(config-line)# exit Exits the VTY line configuration mode.
foo(config)# interface g0/0 Enters interface configuration mode.
foo(config-if)# ip address 192.168.1.2 255.255.255.0 Sets the IP address of the g0/0 interface to
192.168.1.2 with a subnet mask of 255.255.255.0.
foo(config-if)# no shutdown Restarts the interface.
foo(config-if)# exit Exits interface configuration mode.
foo(config)# service password-encryption Encrypts all plaintext passwords.
foo(config)# exit Exits terminal configuration mode.
foo# copy running-config startup-configSaves your configuration to NVRAM.
